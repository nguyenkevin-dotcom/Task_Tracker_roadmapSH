# Task_Tracker_roadmap.sh

*This project has been created by nguyenkevin-dotcom from https://roadmap.sh/projects/task-tracker*

## Description
Project task manager manages existing task by these following commands:

> before every commands it is necessary to write `task-cli`

- **add** "[ *Here write your description* ]"
- **update** [ *id of the task* ] "[ *Here write your description* ]"
- **delete** [ *id of the task* ]
- **list** [*todo | in-progress | done*]
> the arguments for command list are necessary, if type `task-cli list` it will shows every created task.

## Instructions
For program to work you need to type first `pip install -e .` => It will install the project as a package so then you are able to play with it in terminal.

The output of last line should be:
```
Successfully installed task-tracker-0.0.1
```
